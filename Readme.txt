# 📊 Advertising Campaign Analytics Dashboard – Power BI

This project presents an **interactive Power BI dashboard** designed to analyze the performance of an **advertising campaign conducted by an LPAT firm**. The dashboard provides detailed insights into campaign performance, revenue generation, and key business metrics to support data-driven marketing decisions.

🔗 **Live Dashboard Link**: [View on Power BI](https://app.powerbi.com/groups/me/reports/495d3ab6-899e-454b-917e-602c5539fa1d/ReportSection?experience=power-bi)

---

## 📌 Objective

The primary goal of this dashboard is to:

- Track and measure the effectiveness of an advertising campaign.
- Visualize the **revenue generated** from different campaign sources.
- Provide insights into campaign engagement, impressions, and conversion performance.
- Enable stakeholders to **optimize marketing strategies** based on data.

---

## 💼 Key Insights

- **Revenue Analysis**: Compare actual revenue vs expected outcomes.
- **Campaign Performance**: Analyze ad performance by channel, region, and time.
- **Engagement Metrics**: Evaluate click-through rates, impressions, and bounce rates.
- **Return on Ad Spend (ROAS)**: Identify the most and least profitable campaigns.

---

## 📊 Tools & Technologies

- **Power BI** – For data visualization and dashboard development
- **DAX** – For calculated columns and measures
- **Excel / CSV** – As a data source (if applicable)
- **Power Query (M language)** – For data transformation and modeling

---

## 📈 Visual Elements Included

- Line and bar charts for trend analysis
- KPI cards for quick stats
- Interactive slicers for filtering by campaign, time, and region
- Revenue heatmap by source
- ROAS matrix for profitability insights
